## 一、孔安国注（汉）

| 维度 | 解剖内容 |
|------|----------|
| **文本解释** | 「乞之四鄰以應求者」——微生高向四邻乞讨醋来应付求醋者 |
| **方法论** | 训诂+义理结合。先释名（微生姓、高名、鲁人），再判德（用意委曲） |
| **概念重构** | 将「直」定义为「无委曲」——有则说有，无则说无，不绕弯子 |
| **立场假设** | 儒家正统立场：直是核心德目，不容委曲求全 |
| **隐含前提** | 「乞诸邻」本身不是问题，问题在于「应求者」——即隐瞒真相、冒充己有 |
| **时代动机** | 汉初儒学重建期，强调德目标准的清晰性，反对乡愿式调和 |

**冷知识命题**：孔安国已暗示「称邻以与之」则无可厚非，但后世朱熹等只取前半句，忽略此 nuanced 区分。

## 二、刘宝楠《论语正义》（清）

| 维度 | 解剖内容 |
|------|----------|
| **文本解释** | 详考「醯」字训诂（酸、酢、醋），引《仪礼》《周官》《礼记》佐证 |
| **方法论** | 清代考据学典范：音韵+文字+礼制三重证据 |
| **概念重构** | 提出关键区分：「冒为己物以与人」vs「称邻以与之」——前者不直，后者厚德 |
| **立场假设** | 考据家立场：先还原文本原貌，再作价值判断 |
| **隐含前提** | 微生高的动机不可考，只能据行为推断；但行为本身有「曲」与「厚」两种可能 |
| **时代动机** | 清代汉学复兴，以考据纠正宋学义理空疏之弊 |

**冷知识命题**：刘宝楠实际上为微生高「平反」留了后门——若他明言「此邻人之醯」，则非但不直之讥，反成厚德之证。

## 三、朱熹《四书章句集注》（宋）

| 维度 | 解剖内容 |
|------|----------|
| **文本解释** | 「讥其曲意徇物，掠美市恩，不得为直也」 |
| **方法论** | 义理派：以理学心性论解读，强调「诚」为直之本 |
| **概念重构** | 「直」=「诚」的外显，曲意徇物即不诚 |
| **立场假设** | 理学立场：道德判断基于动机（心术），非仅行为 |
| **隐含前提** | 微生高「素有直名」——说明此章是针对「名实不符」的批判 |
| **时代动机** | 宋代理学建构期，通过经典诠释建立心性论体系 |

**冷知识命题**：朱熹引入「掠美市恩」四字，将微生高定性为刻意求名，但原文并无「市恩」证据——这是理学「诛心」传统的典型操作。

## 四、程子、范氏（宋）

| 维度 | 解剖内容 |
|------|----------|
| **文本解释** | 程子：「所枉虽小，害直为大」；范氏：「是曰是，非曰非；有谓有，无谓无，曰直」 |
| **方法论** | 理学格言体：以警句概括义理 |
| **概念重构** | 将「直」逻辑化为「真值对应」——是/非、有/无的如实陈述 |
| **立场假设** | 程朱理学立场：直是「理」的显现，委曲即蔽于欲 |
| **隐含前提** | 微事可以见大——「一介之取予」可见「千驷万钟」之操守 |
| **时代动机** | 宋代士大夫修身文化：以小见大，日用即道 |

**冷知识命题**：范氏的定义「是曰是，非曰非」实际上更接近逻辑学上的「真」，而非伦理学上的「直」——这是宋代理学「逻辑化」儒家德目的隐秘尝试。

## 五、张岱《四书遇》（明）

| 维度 | 解剖内容 |
|------|----------|
| **文本解释** | 「非讥刺微生高也。不然乞邻与人，亦是好事，夫子何刻毒至此？」 |
| **方法论** | 心学直觉：以「良知」判断，反对苛责 |
| **概念重构** | 「直」不必等于「机械如实」——助人亦是直心之发 |
| **立场假设** | 阳明心学立场：动机良善即可，不必拘泥形式 |
| **隐含前提** | 孔子不应如此「刻毒」——质疑主流解读的合理性 |
| **时代动机** | 晚明心学：反对朱学僵化，主张活泼泼的良知 |

**冷知识命题**：张岱是少数为微生高辩护者，但他的辩护基于「夫子不刻毒」的信念，而非文本证据——这是心学「六经注我」的典型。

## 六、钱穆《论语新解》（现代）

| 维度 | 解剖内容 |
|------|----------|
| **文本解释** | 「微生素有直名，孔子从此微小处断其非为直人」 |
| **方法论** | 现代学术：考据+义理+文化心理分析 |
| **概念重构** | 「直」=不委曲世故，微生高「以博取人之称誉」 |
| **立场假设** | 文化史立场：关注「乡愿」式人格的社会心理机制 |
| **隐含前提** | 微生高即尾生高——将「守信」与「直」关联，指出其人格矛盾 |
| **时代动机** | 民国新儒学：以现代学术方法重构经典解释 |

**冷知识命题**：钱穆将微生高与「乡愿」挂钩，但原文并无「乡愿」字样——这是以孔子其他章节（阳货篇）回注此章的「互文」操作。

## 九、核心发现

### 发现一：方法论演进
- **汉学**：训诂为基础，义理为归宿
- **宋学**：义理为先，经典为注脚
- **清学**：考据为据，纠正宋学空疏
- **现代**：多元方法，文化史视角

### 发现二：「直」的概念扩张
- 孔安国：「无委曲」→ 行为层面
- 朱熹：「诚之外显」→ 动机层面
- 阳明/张岱：「良知发用」→ 本体层面

### 发现三：微生高身份之争的影响
- 若微生高=尾生高（守信者），则「守信≠直」
- 若微生高≠尾生高，则只是一个普通案例
