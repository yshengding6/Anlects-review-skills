# Cultural Comparator v2.0.0 — 三层文化比较器

> **适用场景**: 中国传统概念的多维度跨文化比较分析
> **上游Skill**: `literature-review`（可提供核心概念作为比较起点）
> **下游Skill**: `classical-content-writer`（比较结果可用于文章中的跨文化参照段落）

---

## 核心架构：三层比较

本 Skill 采用**三层递进**的比较架构，从内部争论到外部对话到终极追问：

```
第一层：诸子百家横向对话（内部多元）
    ↓
第二层：东西方思想纵深对照（跨文明对话）
    ↓
第三层：宗教传统交叉审视（终极意义维度）
```

**为什么是三层**：
- 只做"中西对比"容易陷入"中国 vs 西方"的二元幻觉，忽略中国内部的思想争鸣
- 只做"诸子百家"又容易在中国话语体系内自说自话，缺少外部参照
- 宗教维度经常被哲学比较忽视，但很多中国概念（天命、性、德）的深层逻辑必须放在宗教语境中才能理解

---

## 第一层：诸子百家横向对话

### 目的

在跨出中国之前，先搞清楚中国内部对这个概念有什么争论。**没有内部争鸣的跨文化比较是空谈。**

### 7家观点速查

| 学派 | 代表 | 核心立场 | 对"人性"的回答 | 对"秩序"的回答 | 对"修养"的回答 |
|------|------|---------|--------------|--------------|--------------|
| 儒（孔） | 孔子 | 仁礼相须 | 性相近，习相远 | 为政以德 | 克己复礼 |
| 儒（孟） | 孟子 | 性善扩充 | 恻隐之心人皆有 | 仁政民本 | 养浩然之气 |
| 儒（荀） | 荀子 | 礼法双轨 | 性恶，其善伪也 | 隆礼重法 | 化性起伪 |
| 道（老） | 老子 | 道法自然 | 自然素朴 | 无为而治 | 致虚守静 |
| 道（庄） | 庄子 | 逍遥齐物 | 天性完整 | 不刻意 | 心斋坐忘 |
| 墨 | 墨子 | 兼爱功利 | 性可教化 | 尚贤尚同 | 苦行节用 |
| 法 | 韩非 | 法术势 | 性恶自为 | 法治集权 | 参验功用 |

### 诸子横向对比方法

对每个核心概念，按以下格式执行7家扫描：

```
[概念] 的诸子分歧：
- 孔子：[立场] —— [原文依据]
- 孟子：[立场] —— [原文依据]
- 荀子：[立场] —— [原文依据]
- 老子：[立场] —— [原文依据]
- 庄子：[立场] —— [原文依据]
- 墨子：[立场] —— [原文依据]
- 韩非：[立场] —— [原文依据]

核心分歧线：[哪几家形成对立？对立的根源是什么？]
共识区：[哪几家意外地站在一起？]
```

**强制规则**：必须找出至少一组"非典型联盟"——比如墨子和韩非在"功利"上的暗合、老子和荀子在"去伪"上的交叉。这种反常识发现是比较的核心价值。

## 第二层：东西方思想纵深对照

### 目的

把中国概念放入世界思想史的坐标系，寻找真正的对话点——不是简单类比，而是找出"同一问题的不同回答"。

### 概念对应映射

将中国先秦概念与西方哲学概念进行系统性对照：

| 中国概念 | 西方对照 | 映射类型 | 核心差异 |
|---------|---------|---------|---------|
| 仁 | phronesis (亚里士多德) / good will (康德) / agape (基督教) | 部分重叠 | 儒家仁是情感发用，西方是理性原则；仁重推恩，西方重普遍 |
| 礼 | nomos (希腊) / Sittlichkeit (黑格尔) / ritual (宗教) | 功能类比 | 礼是情感化规范（礼乐教化），nomos 是理性规范/习俗 |
| 义 | dikaiosyne (柏拉图) / justice (罗尔斯) | 语义偏移 | 儒家义侧重内在情感外发，西方义侧重外在规范/制度 |
| 道 | logos (赫拉克利特) / Tao (翻译借用) / Way (宗教) | 结构对应 | 道是整体性本体，logos 是理性法则，维度不同 |
| 性 | nature (亚里士多德) / human nature (霍布斯/卢梭) | 领域重叠 | 性含道德向度，nature 侧重自然事实 |
| 天命 | Providence (基督教) / Destiny (希腊) / 法 (佛教) | 功能类比 | 天命内化为人性，Providence 是外在神意 |
| 德 | virtue (亚里士多德) / Tugend (康德) | 语义偏移 | 德是得于道者（得道），virtue 是品格卓越 |
| 无为 | ataraxia (伊壁鸠鲁) / gelassenheit (海德格尔) | 部分重叠 | 无为是退隐超越，ataraxia 是理性平静 |

**映射类型说明**：
- **部分重叠**：概念有交集但不完全等同
- **功能类比**：在各自体系中扮演相似角色
- **结构对应**：在思想体系中的位置相似
- **领域重叠**：讨论同一问题域但方法不同
- **语义偏移**：翻译看似对应，实际内涵有系统性差异——**这是最值得关注的一类**

### 五维纵深对比

每个概念从以下5个维度进行纵深比较：

1. **伦理学维度**：道德判断的基础是什么？（关系/义务/后果/品格/神圣命令）
2. **政治学维度**：政治秩序的合法性来源？（天命/契约/自然法/神权/强力）
3. **认识论维度**：知识的来源和标准？（经验/理性/启示/体悟/经典）
4. **人性论维度**：人性的基本预设？（善/恶/白板/自然/堕落）
5. **终极关怀维度**：人生的最高目标是什么？（成圣/解脱/救赎/幸福/自由）

### 对话模拟

在五维对比的基础上，生成一段虚拟对话，让中国哲学家和西方哲学家就同一问题展开交锋：

```
[中国哲学家]：[基于原文立论]
[西方哲学家]：[基于体系回应]
[中国哲学家]：[反诘]
[西方哲学家]：[防御或调整]
→ 不可通约点浮现
```

**对话模拟的铁律**：
- 每位哲学家必须引用自己的核心命题或原文
- 不允许一方轻易被说服——如果一方秒认输，说明比较是表面的
- 必须暴露至少一个"不可通约点"

## 第三层：宗教传统交叉审视

### 目的

很多中国概念（天命、性、德、礼、鬼神）在哲学层面无法穷尽其含义，必须进入宗教语境才能理解其深层逻辑。反过来，西方宗教传统也提供了哲学体系之外的参照。

### 三大宗教传统对照

#### 东亚宗教

| 传统 | 对"天"的理解 | 对"人"的理解 | 对"修行"的理解 | 与儒家的关系 |
|------|------------|------------|--------------|------------|
| 儒教（礼制） | 天命靡常，唯德是辅 | 天地之性人为贵 | 修身齐家治国平天下 | — |
| 道教 | 天道自然 | 性命双修 | 内丹外丹 | 互补/竞争 |
| 中国佛教 | 天是六道之一 | 众生皆有佛性 | 戒定慧 | 融合/批判 |

#### 亚伯拉罕传统

| 传统 | 对"神"的理解 | 对"人"的理解 | 对"救赎"的理解 | 与儒家对照 |
|------|------------|------------|--------------|----------|
| 犹太教 | 唯一神，立约 | 按神形象造 | 守律法 | 天命 vs 神意 |
| 基督教 | 三位一体 | 原罪堕落 | 因信称义 | 性善 vs 原罪 |
| 伊斯兰教 | 真主独一 | 代治者 | 顺从真主 | 顺天 vs 顺主 |

#### 南亚传统

| 传统 | 对"终极"的理解 | 对"人"的理解 | 对"解脱"的理解 | 与儒家对照 |
|------|--------------|------------|--------------|----------|
| 印度教 | 梵我合一 | 轮回主体 | 解脱(moksha) | 天人合一 vs 梵我合一 |
| 佛教（上座部） | 无我 | 五蕴和合 | 涅槃 | 无我 vs 性善 |
| 佛教（大乘） | 真如佛性 | 众生皆可成佛 | 菩萨道 | 人皆可为尧舜 vs 众生皆有佛性 |

### 宗教维度比较方法

对每个概念，追问三个宗教层面的问题：

1. **超越性问题**：这个概念是指向超越性的（天/神/梵），还是内在性的（性/心/德）？或者兼而有之？
2. **救赎/解脱问题**：这个概念在"人如何获得终极救赎"的叙事中扮演什么角色？
3. **仪式/实践问题**：这个概念是否有仪式性的表达？（礼是仪式，德是否也是？天命是否有占卜/祭祀对应？）

**核心发现格式**：
```
[概念] 在宗教维度的特殊发现：
- 儒家的[概念]与[某宗教]的[某概念]在[某维度]上有意想不到的相似：[具体说明]
- 但在[某维度]上存在根本分歧：[具体说明]
- 这意味着：[对理解原概念的影响]
```

## 使用方式

### 独立使用——指定概念

```
请比较"仁"在诸子百家、西方哲学和宗教传统中的位置
```

### 独立使用——指定维度

```
请从伦理学维度比较"义"在儒法两家和康德/边沁之间的异同
```

### 接收 literature-review 输出

```
基于上面5.25章的分析，比较"伐善"概念在中西伦理传统中的位置
```

### 只做其中一层

```
只做"礼"的诸子百家横向对比，不需要中西对照和宗教维度
```

## 执行规则

1. **三层按序执行**：第一层→第二层→第三层，不跳层
2. **用户可指定只做某一层**：如"只做诸子横向对比"
3. **证据锚定**：所有观点必须标注 [实证]（原文支持）/ [推论]（逻辑推导）/ [缺失]（证据不足）
4. **禁止肤浅类比**：不允许"仁=爱"这种一句话对照，必须展开差异
5. **非典型联盟优先**：诸子横向对比中，反常识的联盟比常识的对立更有价值
6. **不可通约性必须暴露**：对话模拟中，如果双方"完全能互相理解"，说明比较不够深
