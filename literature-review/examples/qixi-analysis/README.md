## 📚 项目简介

本项目是对《论语·公冶长》第五章第二十三章「乞醯章」的深度文献综述。仅十九字的经文，却引发了两千多年的学术争论。本项目通过多种分析方法，系统梳理了历代注疏的核心观点、方法论差异及争议焦点，并提出「老丁因子」——一种非共识的观察视角。

> 微生高不是不直，而是「直」这个概念本身就不足以描述人性的复杂。孔子此章，是在邀请我们放下标签，看见具体的人。

## 🎯 核心发现

### 历代注疏争议焦点

| 争议 | 问题 | 对立双方 |
|------|------|---------|
| **争议一** | 微生高是否「不直」？ | 孔安国/朱熹（不直） vs 王阳明/张岱（未必） |
| **争议二** | 「直」的定义是什么？ | 无委曲（汉学） vs 心诚（理学） vs 本心良善（心学） |
| **争议三** | 孔子态度是讥刺还是存疑？ | 讥刺派（主流） vs 存疑派（刘宝楠、张岱） |

### 跨学派差异

| 学派 | 「直」的定义 | 对微生高的评判 |
|------|-------------|---------------|
| **儒家（孔）** | 无委曲，名实相符 | 曲意徇物，非直 |
| **儒家（孟）** | 恻隐之心的自然发用 | 可能肯定其恻隐 |
| **道家（老）** | 大直若屈 | 或正是「大直」 |
| **墨家** | 利人即为直 | 利人行为正当 |
| **法家** | 法的属性，非个人德目 | 无关公法的私义 |

### 东西方差异

| 维度 | 中国 | 西方 |
|------|------|------|
| **本体论** | 「直」与「诚」相连，涉及宇宙论 | 诚实是理性存在的道德义务 |
| **关系性** | 「直」在关系中显现 | 诚实是个体对理性法则的服从 |
| **情境性** | 观人于微，从具体情境判断 | 定言命令，无条件的普遍法则 |

## 💡 老丁因子（非共识观察）

> **「乞醯章」不是在批评微生高，而是在批评那个给他「直名」的社会——以及我们今天还在用「直/不直」二元标签评判他人的习惯。**

### 核心观点

1. **微生高是个「背锅侠」**
   - 历代注家都在分析微生高哪里「不直」，但没人问：谁给他封的「直名」？
   - 孔子问「孰谓微生高直」——重点在「孰谓」，不在「微生高」。

2. **「直」这个概念本身就是陷阱**
   - 儒家讲「直」，道家讲「大直若屈」，墨家讲「利人」，法家讲「法」。
   - 大家都在争什么是「直」，却没人问：为什么一定要「直」？

3. **最讽刺的是注家自己**
   - 朱熹说微生高「掠美市恩」——但朱熹作《集注》，不也是在「掠」孔子之「美」以「市」后世之「恩」吗？

4. **苏格拉底的启示**
   - 「『孰谓微生高直』或许不是一个判断句，而是一个疑问句。」
   - 孔子这一问，可能不是要定罪，而是要**打破我们对「直」的执念**。
